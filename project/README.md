# Структура проекта:

```
project/
├── project_TuranosovKV.ipynb # Основной Jupyter Notebook с выполнением анализа
├── presentation.pdf # Презентация с результатами проекта
├── README.md # Описание проекта и инструкции по запуску
├── environment.yml # Файл с конфигурацией окружения conda
└── requirements.txt # Список зависимостей для pip (если необходимо
```

## Инструкция по запуску

1. **Создайте окружение с помощью файла `environment.yml`:**

    Для развертывания всех зависимостей, используйте команду:
    ```bash
    conda env create -f environment.yml
    ```

2. **Активируйте созданное окружение:**

    После установки зависимостей активируйте окружение:
    ```bash
    conda activate new_big_data
    ```

3. **Запустите Jupyter Notebook:**

    Откройте Jupyter Notebook для работы с проектом:
    ```bash
    jupyter notebook project_TuranosovKV.ipynb
    ```

    Это откроет основной ноутбук, где вы сможете просматривать и запускать анализ.

### Зависимости

- **Python**: 3.11
- **PySpark**: для работы с большими данными
- **OpenJDK 17**: для работы с Java-базированными компонентами, такими как Spark

Проект использует **conda** для установки всех зависимостей. Дополнительно, для пользователей, которые хотят установить зависимости через `pip`, имеется файл `requirements.txt`.
